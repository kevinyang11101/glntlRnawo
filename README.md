## 前言

本项目为校园体育场馆（设施）使用管理网站，是针对校园内体育场馆、设施进行管理的在线平台。通过本项目，可以实现体育场馆的在线预约、使用管理、信息发布等功能，提高校园体育设施的利用率，方便师生使用。以下是本项目的详细内容介绍。

## 内容介绍

本项目基于Java语言，采用Spring Boot框架，结合前端技术JS、Vue和css3，实现了以下功能：

1. 用户注册、登录、个人信息管理
2. 体育场馆信息展示、查询、预约
3. 管理员登录、场馆信息管理、预约审核
4. 系统公告发布、新闻动态展示

此外，项目还具备良好的用户体验，如友好的界面设计、简洁的操作流程、实时反馈等。

## 技术介绍

- 语言：Java
- 使用框架：Spring Boot
- 前端技术：JS、Vue、css3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
